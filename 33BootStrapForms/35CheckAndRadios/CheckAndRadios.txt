Checkboxes
Las casillas de verificación se utilizan si desea que el usuario seleccione cualquier cantidad de opciones de una lista de opciones preestablecidas.

Ejemplo explicado
Para aplicar estilo a las casillas de verificación, utilice un elemento envolvente class="form-check"para garantizar los márgenes adecuados para las etiquetas y las casillas de verificación.

Luego, agregue la .form-check-labelclase para etiquetar elementos y .form-check-inputaplicar estilo a las casillas de verificación correctamente dentro del .form-checkcontenedor.

Utilice el checkedatributo si desea que la casilla de verificación esté marcada de forma predeterminada.
