Barra de progreso básica
Se puede usar una barra de progreso para mostrar qué tan lejos está un usuario en un proceso.
Para crear una barra de progreso predeterminada, agregue una .progressclase a un elemento contenedor y agregue la .progress-barclase a su elemento secundario. Use la propiedad CSS widthpara establecer el ancho de la barra de progreso:

Altura de la barra de progreso



La altura de la barra de progreso es 1rem(generalmente 16px) por defecto. Use la propiedad CSS heightpara cambiarlo. Tenga en cuenta que debe establecer la misma altura para el contenedor de progreso y la barra de progreso:
